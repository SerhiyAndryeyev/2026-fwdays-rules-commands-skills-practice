# Specs Driven Development 2026 Presentation

## Key Ideas:
- SDD is the evolution of TDD (Test Driven Development).
- The Spec (technical requirement) is the "Source of Truth".
- Agentic IDEs don't just suggest code, they execute the "Plan-Action-Verify" loop.

## Tool Highlights:
- **Cursor**: Best for rapid prototyping via Composer.
- **Antigravity**: Superior for custom workflows and BA-to-Dev handoff.
- **Claude Code**: Best for complex CLI-based refactoring.
- **Codex**: Enterprise-grade repository understanding.

## Main Message:
In 2026, a developer's main skill is writing precise specs, not just syntax.
